Bades.id

Kebijakan Privasi

Berlaku sejak: 24 Mei 2026

Patuh UU PDP 27/2022

Bades.id berkomitmen melindungi data pribadi warga desa yang Anda kelola lewat layanan kami. Kebijakan ini menjelaskan data apa yang kami simpan, bagaimana kami pakai, dan hak Anda.

1. Data yang dikumpulkan

Kami mengelola dua jenis data:

  • Data akun operator: nama, email, sandi (hash), waktu login.
  • Data warga & administrasi desa (dimiliki desa Anda): NIK, KK, alamat, tanggal lahir, jenis kelamin, foto, dokumen surat, data keuangan desa, program bantuan, dan informasi terkait yang Anda input ke ruang kerja.

2. Tujuan pemakaian

  • Menjalankan layanan administrasi desa sesuai instruksi Anda.
  • Menjaga keamanan akun dan mencegah penyalahgunaan.
  • Pemeliharaan teknis dan pemulihan layanan.

Kami tidak menjual atau membagi data warga ke pihak ketiga untuk tujuan komersial.

3. Lokasi penyimpanan

Sesuai UU PDP, semua data warga disimpan di server di wilayah Indonesia (AWS Jakarta region). Backup harian otomatis di-encrypt di server yang sama.

4. Pemrosesan & akses

  • Hanya operator yang Anda undang yang dapat mengakses data ruang kerja desa Anda.
  • Tim Bades dapat mengakses data hanya untuk dukungan teknis atas izin tertulis Anda, atau saat diharuskan hukum.
  • NIK dan dokumen identitas sensitif di-masking untuk role non-administratif.

5. Keamanan

  • Sandi disimpan dalam bentuk hash bcrypt, tidak pernah plaintext.
  • Koneksi pengguna dengan server melalui HTTPS (TLS 1.2+).
  • Database terisolasi di subnet privat, tidak terpapar internet.
  • Audit log aktivitas mengakses data sensitif.

6. Retensi data

Data warga dipertahankan selama ruang kerja desa aktif. Setelah desa menutup ruang kerja, data dihapus permanen dalam 30 hari, kecuali Anda meminta ekspor terlebih dahulu.

7. Hak subjek data

Sesuai UU PDP, subjek data (warga) berhak:

  • Mengetahui data pribadinya yang disimpan.
  • Meminta pembetulan data yang salah.
  • Meminta penghapusan data (kecuali wajib disimpan oleh peraturan).
  • Mengajukan keberatan atas pemrosesan.

Permintaan ini diajukan ke desa pengelola ruang kerja, atau ke tim Bades melalui privasi@bades.id.

8. Pelanggaran data

Kami akan memberitahu pengguna terdampak dan otoritas terkait dalam 72 jam setelah pelanggaran data terdeteksi, sesuai amanat UU PDP.

9. Perubahan kebijakan

Versi terbaru kebijakan ini akan diumumkan via email dan diposting di halaman ini.

10. Kontak

Pertanyaan terkait privasi: privasi@bades.id

Petugas Pelindungan Data (DPO): akan ditunjuk sebelum layanan publik luas.
